# Data-Science-ML-Projects/WorldPopulationAnalysis
This repository contains Data Science and Machine Learning projects alongwith detailed analysis ppt and notebook file.

World population Dataset contains Historical Population data for every Country/Territory in the world by different parameters like Area Size of the Country/Territory, 
Name of the Continent, Name of the Capital, Density, Population Growth Rate, Ranking based on Population, World Population Percentage.

The Objective is:
1. To analyze the population trends in various countries and world as a whole over the years.
2. To visualize the same with graphs
3. To analyze the UN population dataset and join with the given dataset to find insights
4. To predict population and correlation between features using Linear Regression
